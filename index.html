CineLog — Web Development Course Project Full-stack movie app with AI-powered recommendations
React + Supabase + AI Integration expand +
The Project

CineLog is a full-stack movie tracking app built for a web development course. Users can search films via the TMDB API, save them to a personal watchlist, and get AI-generated recommendations by describing a mood or vibe. Deployed live on GitHub Pages.

View live project →

What I Built
  • User authentication with session persistence using Supabase Auth and React Context
  • Movie search pulling from the TMDB API, with real-time watchlist status on every card
  • Protected routes that wait for auth state before redirecting
  • AI recommendation engine — user describes a mood, a local LLM streams back four tailored picks in real time
  • User profile page with a separate profiles table and upsert logic for clean data management
Tech Stack & Reflection
  • React + Vite — component architecture, hooks, React Router
  • Supabase — auth, database, real-time queries
  • TMDB API — movie data and search
  • Ollama (Gemma 3:4b) — streaming LLM integration
  • GitHub Pages — deployment via gh-pages

If I rebuilt this, I would replace the local Ollama dependency with a hosted API so the AI feature works on the live deployment for anyone. I would also add loading skeleton UI and better error states throughout.
